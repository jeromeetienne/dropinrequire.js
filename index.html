Drop-In require

dropinrequire.js is a dropin replacement for commonjs module in a browser. No server to run, No preprocessing of javascript files. It aims for If you are using node.js and is willing to share code with your browser without hassle, dropinrequire.js is for you.

How to use it

You just include this in your webpage

<script type="text/javascript" src="./dropin_require.js"></script>

And after that you can use require(filename) to import your node module in the web browser. Suppose you got a module foobar.js

exports.foo = "bar";

And with this html, it will display bar. Show, dont tell? Here it is live

<script src="../dropin_require.js"></script>
<script>
    alert( require('./foobar.js').foo );
</script>
